# Kode.Agent.Boilerplate.React

Modern React frontend for Kode.Agent with ChatGPT-like interface.

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS 4.1
- Shadcn UI

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000

See [README.md](./README.md) for full documentation.
